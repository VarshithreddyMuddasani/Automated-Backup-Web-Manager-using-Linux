
The Automated Backup Web Manager is a Linux-based web application designed to automate and manage system backups efficiently. The project uses Bash scripting, Cron jobs, rsync, and tar to perform automated directory backups and compress them into archive files. A simple PHP-based web interface hosted on Apache allows administrators to view backup logs, monitor backup activities, and download backup archives directly through a web browser. This project demonstrates how Linux automation tools and web technologies can be integrated to create a reliable and user-friendly backup management solution for system administrators and small enterprise environments.
